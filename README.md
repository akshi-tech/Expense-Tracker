# 💰 Expense Tracker

A simple and responsive web application that helps users track and manage their daily expenses. The application allows users to record expenses, categorize transactions, monitor total spending, and view monthly summaries to better understand their spending habits.

## 📌 Features

* Add daily expenses
* Categorize expenses (Food, Travel, Shopping, Bills, Others)
* View expense history
* Calculate total spending
* Monthly expense summary
* Responsive and user-friendly interface
* Data stored using Browser Local Storage

## 🛠️ Technology Stack

* **HTML** – Structure of the application
* **CSS** – Styling and responsive design
* **JavaScript** – Application logic and interactivity
* **Browser Local Storage** – Stores expense data locally without a database

## 🎯 Project Objective

The objective of this project is to help users manage their daily expenses efficiently and develop better financial habits through a simple and easy-to-use interface.
